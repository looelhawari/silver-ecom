# Access Control Module

**Responsibility:** Roles and permissions (Spatie). Defines roles: super-admin, staff, order-manager, content-manager, product-manager and their permissions. Backs Filament panel and API authorization.

**Owned tables (planned):** roles, permissions, model_has_roles, model_has_permissions, role_has_permissions

**Depends on:** —

> Code folders (`Http`, `Models`, `Services`, `Actions`, `Policies`, `Routes`,
> `Http/Requests`, `Http/Resources`, `Tests`) are added in this module's
> implementation phase. See `/MODULES.md` for the full module map and boundary rules.
