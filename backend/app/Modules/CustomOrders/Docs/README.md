# Custom Orders Module

**Responsibility:** Custom silver requests: reference image upload, description, silver type/weight/size, guest tracking code, admin quote workflow (accept/reject/request-details), and convert-to-order.

**Owned tables (planned):** custom_order_requests, custom_order_images, custom_order_quotes

**Depends on:** Catalog, Orders, Media, Users

> Code folders (`Http`, `Models`, `Services`, `Actions`, `Policies`, `Routes`,
> `Http/Requests`, `Http/Resources`, `Tests`) are added in this module's
> implementation phase. See `/MODULES.md` for the full module map and boundary rules.
