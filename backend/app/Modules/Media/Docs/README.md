# Media Module

**Responsibility:** Shared, secure file upload/storage service used by product images, payment proofs, and custom-order images. Validates MIME+extension, limits size, renames files, stores safely off web root.

**Owned tables (planned):** media (optional)

**Depends on:** —

> Code folders (`Http`, `Models`, `Services`, `Actions`, `Policies`, `Routes`,
> `Http/Requests`, `Http/Resources`, `Tests`) are added in this module's
> implementation phase. See `/MODULES.md` for the full module map and boundary rules.
