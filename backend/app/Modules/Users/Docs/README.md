# Users Module

**Responsibility:** Customer and admin user accounts, profiles, saved addresses, admin notes, block/unblock. A customer is a User holding the customer role.

**Owned tables (planned):** users, user_addresses, user_admin_notes

**Depends on:** AccessControl

> Code folders (`Http`, `Models`, `Services`, `Actions`, `Policies`, `Routes`,
> `Http/Requests`, `Http/Resources`, `Tests`) are added in this module's
> implementation phase. See `/MODULES.md` for the full module map and boundary rules.
